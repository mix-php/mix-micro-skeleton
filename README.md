## Mix Micro Skeleton

微服务开发程序骨架，包含微服务开发的大量范例代码

## 开发文档

MixPHP 开发指南：http://doc.mixphp.cn

## 环境要求

* Linux, OS X, WSL
* PHP >= 7.2
* Swoole >= 4.4.4

## 快速开始

使用 [composer](https://www.phpcomposer.com/) 安装：

```shell
composer create-project --prefer-dist mix/mix-micro-skeleton mix ~2.2.0
```
